AMD Radeon RX 580 8GB

The AMD Radeon RX 580 8GB is a mid-range graphics card launched in 2017, built on the Polaris 20 architecture. It features 2304 stream processors, 144 texture units, and 32 ROPs. The card has 8GB of GDDR5 memory, running at 2090 MHz (8 Gbps effective), and a 256-bit memory interface.
